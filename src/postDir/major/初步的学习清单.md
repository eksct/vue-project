## ✅ 模块二：Spring Security + JWT 登录鉴权（第 2 月）

### 📌 学习目标：

- 实现完整权限控制体系，支持角色、接口粒度拦截
    

### 📚 学习内容清单：

-  Spring Security 核心过滤器链、用户认证流程
    
-  JWT 的生成、解析、刷新机制
    
-  基于注解的权限控制（`@PreAuthorize`）
    
-  登出处理、黑名单机制、自动续签
    
-  异常处理统一输出（401、403）
    

### 🧪 实践建议：

-  用户登录模块：JWT + Redis 存储会话信息
    
-  管理员权限系统（区分普通用户/后台管理员）
    
-  鉴权失败统一处理响应格式

-  ~~分区、副本机制、消费者组~~

### 🧪 实践建议：
